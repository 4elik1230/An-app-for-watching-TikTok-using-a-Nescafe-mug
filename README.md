# An-app-for-watching-TikTok-using-a-Nescafe-mug-in-Python-3.10

Эта программа сделана на Python 3.10 и может листать ленту тик тока просто показавшы в веб-камеру кружку Nescafe!

"Как это работает?" - Все сделано очень просто когда вы показываете свою кружку в веб-камеру то программа сравнивает её со скриншотом крушки Nescafe, можно в замен этой крушки вставить свою но она должна быть на прозрачном фоне! И когда программа сравнивает крушку с вашей то там должно быть совпадений больше 15 чтобы программа пролистнула видео. Важно чтобы ваша кружка не была с цветом вашего фона! На пример у вас кружка белого цвета и фон такого же цвета, то программа посчитает фон за то что вы показываете кружку.

Библиотеки для работы программы:
1. open-cv
2. numpy
3. pyautogui
